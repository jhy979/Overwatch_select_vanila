#### vanilla CSS의 불편한 점들을 Scss를 통해 코드 리팩토링을 진행했습니다.
> 1. npm project 로 변경 <br>
> 2. parcel bundler 설치 <br>
> 3. main.css -> main.scss <br>
> 4. main.scss 수정  <br>

#### SCSS의 장점
- 반복문을 통해 코드를 훨씬 간결하게 만들 수 있습니다.
- 중첩을 통해 복잡한 html 구조를 더 간결하게 파악할 수 있습니다.
